Every language is a different way of thinking.

- **C / C++** — 离硬件最近的地方，性能与控制的极致
- **C#** — 优雅的面向对象，.NET 生态
- **Python** — 万能胶水，从脚本到 AI
- **Golang** — 简洁、并发、为云而生
- **Markdown** — 写作者的最佳伴侣
- **Bash** — 和系统对话的方式
