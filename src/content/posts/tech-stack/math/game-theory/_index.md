博弈论研究的是**相互依存**的理性决策：每个人的收益不仅取决于自己的选择，还取决于别人的选择。整门课其实就在回答两个问题——**什么是博弈、如何进行博弈**。

全课最该刻进脑子里的一张图，是用**信息**和**时序**两个维度把博弈切成四块，每一块对应一个均衡解概念：

| | 完全信息 | 不完全信息 |
|---|---|---|
| **静态**（同时决策） | 完全信息静态博弈 → **Nash 均衡**（Nash, 1950/51） | 不完全信息静态博弈 → **贝叶斯 Nash 均衡 BNE**（Harsanyi, 1967） |
| **动态**（先后决策） | 完全信息动态博弈 → **子博弈精炼 Nash 均衡 SPNE**（Selten, 1965） | 不完全信息动态博弈 → **精炼贝叶斯 Nash 均衡 PBE**（Selten, 1975） |
